# Business Analytics Dashboard

## 📌 Objective

The objective of this project is to build an interactive Business Analytics Dashboard that helps visualize and analyze business performance. The application provides insights into sales, revenue, profit, and product performance using charts and data representation.

🚀 Functionality
<details> <summary>Click to view</summary>

The application includes the following features:

Displays key business metrics such as total revenue, profit, orders, and products
Visualizes monthly revenue using line charts
Displays category-wise profit using bar charts
Shows sales data in a structured table format
Provides business insights based on data analysis
Fully responsive UI for different screen sizes
